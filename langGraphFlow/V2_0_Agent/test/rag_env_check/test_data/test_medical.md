# 测试文档 - 内科常见疾病

## 高血压

高血压是一种常见的慢性疾病，是指血压持续升高的状态。正常血压应该低于120/80 mmHg。

### 症状
- 头痛
- 头晕
- 心悸
- 胸闷

### 诊断标准
收缩压 ≥ 140 mmHg 或 舒张压 ≥ 90 mmHg

### 治疗方法
1. 生活方式干预：低盐饮食、适量运动、戒烟限酒
2. 药物治疗：ACE抑制剂、ARB、利尿剂等

## 糖尿病

糖尿病是由于胰岛素分泌不足或作用障碍导致的血糖升高疾病。

### 症状
- 多饮
- 多食
- 多尿
- 体重下降

### 诊断标准
空腹血糖 ≥ 7.0 mmol/L 或 餐后2小时血糖 ≥ 11.1 mmol/L

### 治疗方法
1. 饮食控制
2. 运动疗法
3. 药物治疗：口服降糖药、胰岛素
