外科常见疾病诊断指南

急性阑尾炎诊断指南

概述
急性阑尾炎（Acute Appendicitis）是外科最常见的急腹症，好发于10-30岁，男性略多于女性。

典型症状
- 转移性右下腹痛：初始上腹或脐周疼痛，数小时后转移至右下腹
- 恶心、呕吐：早期出现，多为反射性
- 发热：体温通常37.5-38.5℃，阑尾穿孔时可更高
- 食欲不振

体征检查
- 右下腹压痛：麦氏点（McBurney点）压痛最明显
- 反跳痛：提示腹膜刺激征
- 肌紧张：腹壁肌肉紧张
- 结肠充气试验（Rovsing征）：阳性
- 腰大肌试验：阳性提示阑尾位于盲肠后位
- 闭孔内肌试验：阳性提示阑尾靠近闭孔内肌

辅助检查
1. 血常规：白细胞计数升高（>10×10⁹/L），中性粒细胞比例升高
2. C反应蛋白（CRP）：升高
3. 腹部B超：可见阑尾增粗、周围积液
4. CT检查：阑尾增粗、壁增厚、周围脂肪间隙模糊

诊断要点
1. 典型的转移性右下腹痛
2. 右下腹固定压痛
3. 血白细胞升高
4. 影像学检查支持

鉴别诊断
- 右侧输尿管结石：绞痛，血尿，B超可见结石
- 右侧卵巢囊肿蒂扭转：妇科检查，B超
- 急性胃肠炎：腹泻明显，无固定压痛
- 肠系膜淋巴结炎：多见于儿童，压痛不固定
- 右侧肺炎：胸片可见炎症

治疗原则
1. 手术治疗：阑尾切除术（首选）
   - 腹腔镜阑尾切除术（推荐）
   - 开腹阑尾切除术
2. 术前准备：禁食、补液、抗生素
3. 术后处理：抗感染、早期下床活动

并发症
- 阑尾穿孔
- 腹膜炎
- 阑尾周围脓肿
- 门静脉炎

手术指征
- 确诊急性阑尾炎
- 疑似阑尾炎，症状持续加重
- 阑尾穿孔或形成脓肿

骨折诊断指南

概述
骨折是骨的完整性和连续性中断，是外科常见疾病。

分类
- 按原因：外伤性、病理性
- 按程度：完全性、不完全性
- 按形态：横形、斜形、螺旋形、粉碎性

常见症状
- 疼痛：局部剧痛，活动时加重
- 肿胀：局部肿胀、瘀斑
- 畸形：肢体变形、短缩
- 功能障碍：不能活动
- 异常活动：假关节活动
- 骨擦音：可闻及骨摩擦音

体征检查
- 视诊：畸形、肿胀、瘀斑
- 触诊：压痛、骨擦感
- 动诊：异常活动、功能障碍
- 量诊：长度、角度测量

辅助检查
1. X线检查：明确骨折类型、移位情况
2. CT检查：复杂骨折、关节内骨折
3. MRI：软组织损伤、隐匿性骨折

诊断要点
1. 外伤史
2. 典型症状和体征
3. X线检查确诊

治疗原则
1. 复位：手法复位或手术复位
2. 固定：外固定或内固定
3. 功能锻炼：早期、循序渐进

并发症
- 感染
- 骨不连
- 畸形愈合
- 关节僵硬
- 创伤性关节炎
