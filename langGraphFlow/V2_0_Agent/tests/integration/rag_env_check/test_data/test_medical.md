# 内科常见疾病诊断指南

## 高血压诊断指南

### 概述
高血压（Hypertension）是指血压持续升高的慢性疾病，是心血管疾病的重要危险因素。

### 诊断标准
1. **正常血压**：收缩压 < 120 mmHg 且 舒张压 < 80 mmHg
2. **高血压前期**：收缩压 120-139 mmHg 或 舒张压 80-89 mmHg
3. **1级高血压**：收缩压 140-159 mmHg 或 舒张压 90-99 mmHg
4. **2级高血压**：收缩压 ≥ 160 mmHg 或 舒张压 ≥ 100 mmHg

### 常见症状
- 头痛，特别是早晨起床时
- 头晕、眩晕
- 心悸、胸闷
- 视力模糊
- 耳鸣
- 疲劳、乏力

### 诊断要点
1. 至少测量3次不同时间的血压
2. 排除白大衣高血压
3. 进行24小时动态血压监测（必要时）
4. 评估靶器官损害情况

### 鉴别诊断
需要排除继发性高血压：
- 肾性高血压
- 内分泌性高血压
- 药物性高血压

### 治疗原则
1. 生活方式干预：低盐饮食、适量运动、控制体重
2. 药物治疗：ACEI、ARB、CCB、利尿剂等
3. 定期监测血压，调整治疗方案

## 糖尿病诊断与治疗

### 概述
糖尿病（Diabetes Mellitus）是一组以高血糖为特征的代谢性疾病，主要分为1型和2型糖尿病。

### 诊断标准
满足以下任一标准即可诊断：
1. 空腹血糖 ≥ 7.0 mmol/L
2. 随机血糖 ≥ 11.1 mmol/L
3. OGTT 2小时血糖 ≥ 11.1 mmol/L
4. HbA1c ≥ 6.5%

### 常见症状
- 多饮、多尿、多食
- 体重下降
- 视力模糊
- 疲劳、乏力
- 伤口愈合缓慢
- 反复感染

### 1型糖尿病特点
- 多见于青少年
- 起病急，症状明显
- 胰岛素绝对缺乏
- 需要胰岛素治疗

### 2型糖尿病特点
- 多见于中老年
- 起病缓慢，症状不明显
- 胰岛素抵抗为主
- 可通过口服药物控制

### 并发症
- 糖尿病肾病
- 糖尿病视网膜病变
- 糖尿病神经病变
- 糖尿病足
- 心血管疾病

### 治疗目标
- 空腹血糖：4.4-7.0 mmol/L
- 餐后2小时血糖：< 10.0 mmol/L
- HbA1c：< 7.0%

### 治疗方案
1. 生活方式干预：饮食控制、运动、体重管理
2. 口服降糖药：二甲双胍、磺脲类、DPP-4抑制剂等
3. 胰岛素治疗：适用于1型糖尿病和部分2型糖尿病
