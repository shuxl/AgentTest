# 子宫肌瘤诊断指南

## 概述
子宫肌瘤是女性生殖系统最常见的良性肿瘤，好发于30-50岁女性。

## 分类
- **按部位**：肌壁间、浆膜下、黏膜下
- **按数量**：单发、多发

## 常见症状
- **月经改变**：月经过多、经期延长、不规则出血
- **腹部包块**：肌瘤较大时可触及
- **压迫症状**：尿频、便秘、腰酸
- **疼痛**：下腹痛、痛经
- **不孕**：影响受孕或导致流产

## 体征检查
- **妇科检查**：子宫增大、表面不规则、质硬
- **B超**：可见肌瘤大小、位置、数量
- **MRI**：明确肌瘤与周围组织关系

## 辅助检查
1. **B超**：首选检查方法
2. **MRI**：复杂病例
3. **宫腔镜**：黏膜下肌瘤
4. **腹腔镜**：浆膜下肌瘤

## 诊断要点
1. 症状和体征
2. B超检查
3. 排除恶性病变

## 鉴别诊断
- 子宫腺肌病：痛经明显，B超可见
- 卵巢肿瘤：附件区包块
- 妊娠：HCG阳性
- 子宫内膜癌：不规则出血，诊刮确诊

## 治疗原则
1. **观察随访**：无症状、肌瘤小
2. **药物治疗**：
   - 促性腺激素释放激素激动剂
   - 米非司酮
3. **手术治疗**：
   - 肌瘤剔除术（保留子宫）
   - 子宫切除术
4. **介入治疗**：子宫动脉栓塞术

## 手术指征
- 月经过多导致贫血
- 肌瘤引起压迫症状
- 肌瘤快速增大
- 怀疑恶变
- 不孕或反复流产

## 预后
- 良性肿瘤，预后良好
- 术后可能复发
- 恶变率低（<0.5%）
