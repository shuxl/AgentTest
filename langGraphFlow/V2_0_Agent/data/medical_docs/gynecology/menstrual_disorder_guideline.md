# 月经失调诊断指南

## 概述
月经失调是妇科常见疾病，包括月经周期、经期、经量的异常。

## 分类
- **月经周期异常**：月经频发、月经稀发、闭经
- **经期异常**：经期延长、经期缩短
- **经量异常**：月经过多、月经过少
- **不规则出血**：功能失调性子宫出血

## 常见症状
- **周期异常**：周期<21天或>35天
- **经期异常**：经期<3天或>7天
- **经量异常**：经量过多或过少
- **不规则出血**：非经期出血
- **伴随症状**：腹痛、贫血、不孕

## 病因
- **内分泌因素**：下丘脑-垂体-卵巢轴功能异常
- **器质性病变**：子宫肌瘤、子宫内膜息肉、肿瘤
- **全身性疾病**：血液病、肝病、肾病
- **药物因素**：激素类药物、抗凝药物

## 辅助检查
1. **妇科检查**：了解子宫、附件情况
2. **B超**：了解子宫、附件、内膜情况
3. **性激素检查**：FSH、LH、E2、P、T、PRL
4. **甲状腺功能**：TSH、T3、T4
5. **血常规**：评估贫血程度
6. **诊断性刮宫**：了解内膜情况

## 诊断要点
1. 详细的月经史
2. 体格检查和妇科检查
3. 辅助检查明确病因

## 鉴别诊断
- 妊娠相关：异位妊娠、流产
- 器质性病变：子宫肌瘤、子宫内膜癌
- 内分泌疾病：多囊卵巢综合征、甲状腺疾病
- 血液病：血小板减少、凝血功能障碍

## 治疗原则
1. **病因治疗**：针对原发病治疗
2. **激素治疗**：
   - 调整月经周期
   - 止血
   - 促排卵
3. **手术治疗**：器质性病变需手术
4. **对症支持**：纠正贫血、预防感染

## 预防
- 规律作息
- 合理饮食
- 适当运动
- 避免过度节食
- 定期体检
