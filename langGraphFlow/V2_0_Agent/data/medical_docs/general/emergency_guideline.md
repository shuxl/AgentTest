# 急症初步处理指南

## 概述
本文档提供常见急症的初步诊断和处理原则，帮助进行紧急处理。

## 急性胸痛
### 常见原因
- 急性心肌梗死
- 主动脉夹层
- 肺栓塞
- 气胸

### 初步处理
1. 立即评估生命体征
2. 吸氧、建立静脉通道
3. 心电图、心肌酶检查
4. 必要时紧急转诊

## 急性腹痛
### 常见原因
- 急性阑尾炎
- 急性胆囊炎
- 急性胰腺炎
- 肠梗阻

### 初步处理
1. 评估生命体征
2. 禁食、补液
3. 腹部检查、辅助检查
4. 必要时紧急转诊

## 呼吸困难
### 常见原因
- 急性左心衰
- 哮喘急性发作
- 气胸
- 肺栓塞

### 初步处理
1. 保持呼吸道通畅
2. 吸氧
3. 评估严重程度
4. 必要时紧急转诊

## 意识障碍
### 常见原因
- 脑血管意外
- 低血糖
- 中毒
- 癫痫

### 初步处理
1. 评估生命体征
2. 保持呼吸道通畅
3. 快速血糖检查
4. 紧急转诊

## 处理原则
1. **生命第一**：优先处理危及生命的情况
2. **快速评估**：快速判断严重程度
3. **及时转诊**：复杂或危重病例及时转诊
4. **记录完整**：详细记录处理过程
