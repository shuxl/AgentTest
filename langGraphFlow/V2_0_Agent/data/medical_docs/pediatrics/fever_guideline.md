# 儿童发热诊断指南

## 概述
发热是儿童最常见的症状之一，体温超过37.5℃（腋温）或38℃（肛温）定义为发热。

## 发热分级
- **低热**：37.5-38℃
- **中等热**：38.1-39℃
- **高热**：39.1-41℃
- **超高热**：>41℃

## 常见原因
- **感染性**：病毒、细菌、真菌感染
- **非感染性**：风湿性疾病、肿瘤、药物热

## 常见症状
- 体温升高
- 畏寒、寒战
- 头痛、全身不适
- 食欲不振
- 烦躁或精神萎靡

## 诊断要点
1. 测量体温（腋温、肛温、口温）
2. 询问发热时间、热型、伴随症状
3. 体格检查：咽部、肺部、腹部、皮肤
4. 辅助检查：血常规、CRP、病原学检查

## 辅助检查
1. **血常规**：白细胞、中性粒细胞、淋巴细胞
2. **CRP**：细菌感染时升高
3. **病原学检查**：血培养、咽拭子、尿培养
4. **影像学**：胸片（怀疑肺炎时）

## 鉴别诊断
- 上呼吸道感染：咽痛、流涕、咳嗽
- 肺炎：咳嗽、气促、肺部啰音
- 尿路感染：尿频、尿急、尿痛
- 脑膜炎：头痛、呕吐、颈强直

## 治疗原则
1. **物理降温**：
   - 温水擦浴
   - 减少衣物
   - 多饮水
2. **药物降温**：
   - 对乙酰氨基酚（>3个月）
   - 布洛芬（>6个月）
3. **病因治疗**：抗感染等
4. **对症支持**：补液、休息

## 注意事项
- 3个月以下婴儿发热需立即就医
- 高热惊厥史患儿需及时降温
- 观察精神状态、呼吸、循环
- 避免过度包裹

## 危险信号
- 持续高热不退
- 精神萎靡、嗜睡
- 呼吸困难
- 皮疹、出血点
- 惊厥
