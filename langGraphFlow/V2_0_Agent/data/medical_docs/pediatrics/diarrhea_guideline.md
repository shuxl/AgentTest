# 儿童腹泻诊断指南

## 概述
腹泻是儿童常见疾病，表现为大便次数增多、性状改变，是5岁以下儿童死亡的主要原因之一。

## 分类
- **按病程**：急性（<2周）、迁延性（2周-2个月）、慢性（>2个月）
- **按病因**：感染性、非感染性

## 常见病原体
- **病毒**：轮状病毒、诺如病毒、腺病毒
- **细菌**：大肠杆菌、沙门菌、志贺菌
- **寄生虫**：阿米巴、贾第鞭毛虫

## 常见症状
- **腹泻**：大便次数增多、性状改变（稀水样、黏液便、血便）
- **呕吐**：常见
- **发热**：感染性腹泻多见
- **脱水**：口渴、尿少、眼窝凹陷、皮肤弹性差
- **腹痛**：阵发性腹痛

## 脱水评估
- **轻度**：失水量<5%体重，精神稍差，皮肤弹性正常
- **中度**：失水量5-10%体重，精神萎靡，皮肤弹性差，眼窝凹陷
- **重度**：失水量>10%体重，精神极差，皮肤弹性极差，休克

## 辅助检查
1. **大便常规**：白细胞、红细胞、脂肪球
2. **大便培养**：明确病原体
3. **血常规**：白细胞、中性粒细胞
4. **电解质**：评估脱水程度

## 诊断要点
1. 大便次数增多、性状改变
2. 伴随症状（发热、呕吐、脱水）
3. 病原学检查

## 鉴别诊断
- 生理性腹泻：生长发育正常，无其他症状
- 乳糖不耐受：与乳制品相关
- 肠套叠：血便、腹痛、腹部包块
- 坏死性小肠结肠炎：早产儿，血便、腹胀

## 治疗原则
1. **补液治疗**（最重要）：
   - 口服补液盐（ORS）
   - 静脉补液（重度脱水）
2. **继续喂养**：
   - 母乳喂养继续
   - 配方奶可稀释
   - 避免禁食
3. **药物治疗**：
   - 抗生素（细菌性）
   - 益生菌
   - 蒙脱石散
4. **对症支持**：
   - 退热
   - 止吐

## 预防
- 注意手卫生
- 食物卫生
- 接种轮状病毒疫苗
- 母乳喂养

## 危险信号
- 重度脱水
- 持续高热
- 血便
- 精神萎靡
- 不能口服补液
