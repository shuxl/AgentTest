# 儿童肺炎诊断指南

## 概述
肺炎是儿童常见的下呼吸道感染，是5岁以下儿童死亡的主要原因之一。

## 常见病原体
- **病毒**：呼吸道合胞病毒、腺病毒、流感病毒
- **细菌**：肺炎链球菌、金黄色葡萄球菌、流感嗜血杆菌
- **支原体**：肺炎支原体（学龄期儿童常见）

## 常见症状
- **发热**：中高热多见
- **咳嗽**：早期干咳，后期有痰
- **气促**：呼吸频率增快
- **呼吸困难**：鼻翼扇动、三凹征
- **精神萎靡**：食欲不振、烦躁不安

## 体征检查
- **呼吸频率**：
  - <2个月：>60次/分
  - 2-12个月：>50次/分
  - 1-5岁：>40次/分
- **肺部听诊**：湿性啰音、哮鸣音
- **三凹征**：胸骨上窝、锁骨上窝、肋间隙凹陷
- **发绀**：严重时出现

## 辅助检查
1. **血常规**：白细胞、中性粒细胞、CRP
2. **胸片**：肺部浸润影、实变
3. **病原学**：咽拭子、血培养、痰培养
4. **血气分析**：评估氧合和酸碱平衡

## 诊断标准
满足以下条件：
1. 发热、咳嗽、气促等症状
2. 呼吸频率增快
3. 肺部听诊异常或胸片异常

## 严重程度评估
- **轻度**：门诊治疗
- **重度**：需住院，满足以下任一条件：
  - 呼吸频率明显增快
  - 严重呼吸困难
  - 发绀
  - 意识障碍
  - 拒食或脱水

## 鉴别诊断
- 支气管炎：无气促，无肺部实变
- 哮喘：反复发作，哮鸣音明显
- 肺结核：慢性病程，PPD阳性
- 异物吸入：突然起病，单侧体征

## 治疗原则
1. **抗感染治疗**：
   - 病毒性：对症支持
   - 细菌性：抗生素（青霉素类、头孢类）
   - 支原体：大环内酯类
2. **对症支持**：
   - 退热
   - 止咳化痰
   - 吸氧（必要时）
   - 补液
3. **重症处理**：
   - 住院治疗
   - 氧疗
   - 呼吸支持

## 预防
- 接种疫苗（肺炎疫苗、流感疫苗）
- 避免接触感染源
- 增强免疫力
- 母乳喂养
