# 儿童佝偻病诊断指南

## 概述
佝偻病是儿童期因维生素D缺乏导致的钙磷代谢障碍和骨组织钙化不良。

## 病因
- **维生素D缺乏**：日光照射不足、摄入不足
- **钙缺乏**：摄入不足、吸收障碍
- **其他**：肝肾功能异常、遗传因素

## 常见症状
- **神经精神症状**：烦躁、夜惊、多汗、枕秃
- **骨骼改变**：
  - 头部：方颅、前囟闭合延迟
  - 胸部：鸡胸、漏斗胸、肋骨串珠
  - 四肢：O型腿、X型腿、手镯征、脚镯征
  - 脊柱：后凸或侧弯
- **肌肉改变**：肌张力低下、腹部膨隆（蛙腹）

## 体征检查
- **头部**：方颅、前囟大、闭合延迟
- **胸部**：鸡胸、漏斗胸、肋骨串珠、肋膈沟
- **四肢**：O型腿、X型腿、手镯征、脚镯征
- **脊柱**：后凸、侧弯

## 辅助检查
1. **血生化**：
   - 25-(OH)D降低
   - 血钙降低或正常
   - 血磷降低
   - 碱性磷酸酶升高
2. **X线检查**：
   - 长骨干骺端增宽、杯口状
   - 骨质疏松
   - 骨皮质变薄

## 诊断标准
满足以下条件：
1. 有维生素D缺乏的病因
2. 典型的临床表现
3. 血生化异常
4. X线异常

## 严重程度分级
- **轻度**：仅有神经精神症状，无骨骼改变
- **中度**：有骨骼改变，但无畸形
- **重度**：有明显骨骼畸形

## 鉴别诊断
- 软骨发育不全：遗传性疾病，身材矮小
- 成骨不全：易骨折，蓝巩膜
- 低磷性佝偻病：血磷明显降低，遗传性

## 治疗原则
1. **维生素D治疗**：
   - 口服维生素D制剂
   - 严重时肌注
2. **钙剂补充**：
   - 同时补充钙剂
   - 保证钙摄入充足
3. **日光照射**：
   - 增加户外活动
   - 保证充足日照
4. **对症处理**：
   - 纠正骨骼畸形（严重时手术）

## 预防
- 孕期补充维生素D
- 母乳喂养或配方奶喂养
- 及时添加辅食
- 增加户外活动
- 补充维生素D制剂

## 预后
- 早期诊断和治疗预后良好
- 严重骨骼畸形可能遗留
