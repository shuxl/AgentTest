# 儿童哮喘诊断指南

## 概述
哮喘是儿童常见的慢性气道疾病，以可逆性气流受限和气道高反应性为特征。

## 危险因素
- 家族史：父母有哮喘或过敏史
- 过敏体质：湿疹、过敏性鼻炎
- 环境因素：过敏原、空气污染、被动吸烟
- 感染：病毒感染可诱发

## 常见症状
- **喘息**：呼气性呼吸困难，可闻及哮鸣音
- **咳嗽**：夜间和清晨明显，干咳为主
- **胸闷**：胸部紧迫感
- **气促**：活动后加重

## 体征检查
- **听诊**：双肺哮鸣音，呼气相延长
- **呼吸频率**：增快
- **三凹征**：严重时出现
- **发绀**：严重时出现

## 辅助检查
1. **肺功能**：FEV1、PEF（>5岁可配合）
2. **过敏原检测**：皮肤点刺、血清特异性IgE
3. **胸片**：过度充气，排除其他疾病
4. **血气分析**：评估严重程度

## 诊断标准
满足以下条件：
1. 反复发作的喘息、咳嗽、胸闷、气促
2. 发作时双肺可闻及哮鸣音
3. 症状可自行缓解或治疗后缓解
4. 排除其他疾病

## 严重程度分级
- **间歇发作**：症状<1次/周
- **轻度持续**：症状≥1次/周但<1次/天
- **中度持续**：每日有症状
- **重度持续**：每日有症状，频繁发作

## 鉴别诊断
- 支气管炎：无反复发作，无家族史
- 异物吸入：突然起病，单侧体征
- 肺结核：慢性病程，PPD阳性
- 心源性哮喘：心脏疾病，湿性啰音

## 治疗原则
1. **控制药物**（长期使用）：
   - 吸入糖皮质激素
   - 白三烯调节剂
   - 长效β2受体激动剂
2. **缓解药物**（急性发作）：
   - 短效β2受体激动剂
   - 抗胆碱能药物
3. **避免触发因素**：
   - 避免过敏原
   - 避免感染
   - 避免剧烈运动

## 急性发作处理
1. 立即使用快速缓解药物
2. 评估严重程度
3. 必要时就医
4. 氧疗（严重时）

## 预防
- 避免触发因素
- 规律用药
- 定期随访
- 接种疫苗
