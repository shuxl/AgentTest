# 心力衰竭诊断指南

## 概述
心力衰竭是各种心脏疾病导致心功能不全的综合征，是心血管疾病的终末阶段。

## 分类
- **按部位**：左心衰、右心衰、全心衰
- **按病程**：急性、慢性
- **按射血分数**：HFrEF（射血分数降低）、HFpEF（射血分数保留）

## 常见症状
- **左心衰**：
  - 呼吸困难：劳力性、夜间阵发性、端坐呼吸
  - 咳嗽、咳痰：白色泡沫痰、粉红色泡沫痰
  - 乏力、活动耐量下降
- **右心衰**：
  - 水肿：下肢、全身
  - 腹胀、食欲不振
  - 颈静脉怒张
  - 肝大、腹水

## 体征检查
- **心脏**：心脏扩大、奔马律、杂音
- **肺部**：湿性啰音、哮鸣音
- **其他**：颈静脉怒张、肝大、水肿

## 辅助检查
1. **BNP/NT-proBNP**：诊断和评估严重程度
2. **心电图**：心律失常、心肌缺血
3. **胸片**：心影增大、肺淤血
4. **心脏彩超**：评估心功能、结构
5. **心脏MRI**：评估心肌病变

## 诊断标准
满足以下条件：
1. 心衰的症状和体征
2. 心脏彩超显示心功能不全
3. BNP/NT-proBNP升高

## 严重程度分级
- **NYHA I级**：活动不受限
- **NYHA II级**：轻度活动受限
- **NYHA III级**：明显活动受限
- **NYHA IV级**：休息时也有症状

## 鉴别诊断
- 慢性阻塞性肺疾病：肺功能异常
- 肝硬化：肝功能异常、门脉高压
- 肾病综合征：大量蛋白尿
- 心包疾病：心包积液、缩窄性心包炎

## 治疗原则
1. **一般治疗**：
   - 休息、限盐、限水
   - 吸氧
2. **药物治疗**：
   - ACEI/ARB：改善预后
   - β受体阻滞剂：改善预后
   - 利尿剂：减轻水肿
   - 强心剂：地高辛
   - 醛固酮拮抗剂：改善预后
3. **器械治疗**：
   - 心脏再同步化治疗（CRT）
   - 植入式除颤器（ICD）
4. **手术治疗**：心脏移植

## 预防
- 控制原发病
- 控制危险因素
- 规律用药
- 定期随访
