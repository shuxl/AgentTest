# 心律失常诊断指南

## 概述
心律失常是心脏冲动的频率、节律、起源部位、传导速度异常。

## 分类
- **按频率**：快速性、缓慢性
- **按起源**：室上性、室性
- **按严重程度**：良性、恶性

## 常见类型
- **房性早搏**：提前出现的P波
- **室性早搏**：提前出现的宽大QRS波
- **房颤**：P波消失，f波出现
- **室上性心动过速**：心率150-250次/分
- **室性心动过速**：宽大QRS波，心率>100次/分
- **房室传导阻滞**：PR间期延长或QRS脱落

## 常见症状
- **心悸**：心跳感
- **胸闷**：胸部不适
- **头晕**：脑供血不足
- **黑矇、晕厥**：严重时出现
- **乏力**：活动耐量下降

## 辅助检查
1. **心电图**：诊断金标准
2. **动态心电图**：24小时监测
3. **心脏电生理检查**：明确机制
4. **心脏彩超**：评估心功能

## 诊断要点
1. 症状和体征
2. 心电图检查
3. 明确类型和严重程度

## 治疗原则
1. **药物治疗**：
   - 抗心律失常药物
   - 控制心室率
   - 抗凝（房颤）
2. **非药物治疗**：
   - 电复律
   - 射频消融
   - 起搏器植入
   - 除颤器植入

## 危险评估
- 评估血流动力学影响
- 评估猝死风险
- 评估基础心脏病
