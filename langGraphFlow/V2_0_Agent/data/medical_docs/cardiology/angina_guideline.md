# 心绞痛诊断指南

## 概述
心绞痛是冠状动脉供血不足导致的心肌缺血性疼痛，是冠心病的主要表现。

## 分类
- **稳定型心绞痛**：诱因、性质、持续时间相对固定
- **不稳定型心绞痛**：症状加重、新发、静息时发作

## 典型症状
- **胸痛**：
  - 部位：胸骨后、心前区
  - 性质：压迫感、紧缩感、闷痛
  - 放射：左肩、左臂、下颌、后背
  - 持续时间：3-5分钟，不超过15分钟
- **诱因**：劳累、情绪激动、饱餐、寒冷
- **缓解**：休息、含服硝酸甘油

## 体征检查
- **一般情况**：血压、心率、心律
- **心脏听诊**：心音、杂音、奔马律
- **其他**：颈静脉怒张、肺部啰音

## 辅助检查
1. **心电图**：
   - 静息心电图：ST段压低、T波倒置
   - 运动负荷试验：诱发缺血
2. **心肌酶**：肌钙蛋白、CK-MB
3. **冠脉CTA**：无创检查
4. **冠脉造影**：金标准

## 诊断要点
1. 典型的胸痛症状
2. 心电图异常
3. 冠脉造影证实

## 鉴别诊断
- 急性心肌梗死：疼痛更剧烈、持续时间更长、心肌酶升高
- 主动脉夹层：撕裂样疼痛、血压异常
- 肺栓塞：呼吸困难、低氧血症
- 胃食管反流：与体位相关、抗酸有效

## 治疗原则
1. **急性发作**：
   - 休息
   - 含服硝酸甘油
   - 吸氧
2. **长期治疗**：
   - 抗血小板：阿司匹林、氯吡格雷
   - 他汀类：降脂、稳定斑块
   - β受体阻滞剂：减慢心率
   - 硝酸酯类：扩张血管
   - ACEI/ARB：改善预后
3. **介入治疗**：PCI（经皮冠状动脉介入）
4. **手术治疗**：CABG（冠状动脉旁路移植）

## 危险分层
- **低危**：稳定型，症状轻
- **中危**：症状加重
- **高危**：不稳定型，需紧急处理

## 预防
- 控制危险因素：高血压、高血脂、糖尿病
- 戒烟限酒
- 合理饮食
- 适当运动
- 规律用药
