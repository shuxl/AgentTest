# 乳腺癌诊断指南

## 概述
乳腺癌（Breast Cancer）是女性最常见的恶性肿瘤，好发于40-60岁女性。

## 危险因素
- 年龄：40岁以上
- 家族史：一级亲属有乳腺癌史
- 激素因素：初潮早、绝经晚、未生育、未哺乳
- 既往乳腺疾病：不典型增生
- 遗传因素：BRCA1/BRCA2基因突变

## 常见症状
- **乳房肿块**：无痛性、质硬、边界不清
- **乳头溢液**：血性溢液
- **皮肤改变**：酒窝征、橘皮样改变
- **乳头改变**：乳头内陷、抬高
- **腋窝淋巴结肿大**

## 体征检查
- **乳房触诊**：肿块大小、位置、质地、活动度
- **淋巴结检查**：腋窝、锁骨上、锁骨下
- **皮肤检查**：皮肤改变、溃疡
- **乳头检查**：溢液、内陷

## 辅助检查
1. **乳腺B超**：首选检查，可区分囊实性
2. **乳腺X线（钼靶）**：40岁以上女性筛查
3. **MRI**：高危人群、多中心病灶
4. **病理检查**：细针穿刺、粗针穿刺、手术活检
5. **肿瘤标志物**：CA15-3、CEA

## 诊断要点
1. 乳房无痛性肿块
2. 影像学检查提示恶性
3. 病理检查确诊

## 临床分期（TNM）
- **T**：原发肿瘤大小
- **N**：淋巴结转移情况
- **M**：远处转移

## 病理类型
- 浸润性导管癌（最常见）
- 浸润性小叶癌
- 导管原位癌
- 小叶原位癌

## 治疗原则
1. **手术治疗**：
   - 保乳手术（早期）
   - 改良根治术
   - 全乳切除术
2. **辅助治疗**：
   - 化疗
   - 放疗
   - 内分泌治疗
   - 靶向治疗

## 手术指征
- 可手术的乳腺癌
- 患者身体状况良好
- 无远处转移

## 预后因素
- 肿瘤大小
- 淋巴结转移情况
- 组织学分级
- 激素受体状态
- HER2表达

## 随访
- 术后2年内：每3-6个月
- 术后3-5年：每6-12个月
- 术后5年以上：每年
