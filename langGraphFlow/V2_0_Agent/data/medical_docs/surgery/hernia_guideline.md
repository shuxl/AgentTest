# 腹股沟疝诊断指南

## 概述
腹股沟疝（Inguinal Hernia）是腹腔内脏器通过腹股沟区的薄弱点突出形成的疝，是最常见的腹外疝。

## 分类
- **斜疝**：疝囊从腹股沟管内环突出，经腹股沟管，从外环突出
- **直疝**：疝囊从腹股沟三角（Hesselbach三角）突出
- **股疝**：疝囊从股管突出（较少见）

## 常见症状
- **腹股沟区肿块**：站立、咳嗽、用力时出现，平卧时可消失
- **坠胀感**：肿块处有坠胀不适
- **疼痛**：肿块较大时可引起疼痛
- **消化不良**：肠管进入疝囊时可出现

## 体征检查
- **肿块**：腹股沟区可复性肿块
- **咳嗽冲击感**：手指放在外环处，咳嗽时有冲击感
- **回纳试验**：肿块可回纳入腹腔
- **外环口扩大**：斜疝时外环口可扩大

## 辅助检查
1. **B超**：可见疝囊内容物
2. **CT**：明确疝的类型和内容物
3. **疝造影**：较少使用

## 诊断要点
1. 腹股沟区可复性肿块
2. 咳嗽冲击感阳性
3. 肿块可回纳

## 鉴别诊断
- 睾丸鞘膜积液：透光试验阳性，不能回纳
- 精索静脉曲张：站立时明显，平卧时减轻
- 腹股沟淋巴结肿大：固定，不能回纳
- 髂腰肌脓肿：有感染征象

## 治疗原则
1. **手术治疗**（主要方法）：
   - 传统疝修补术
   - 无张力疝修补术（推荐）
   - 腹腔镜疝修补术
2. **保守治疗**：仅适用于不能手术的患者（使用疝带）

## 手术指征
- 所有可复性腹股沟疝
- 嵌顿疝（急诊手术）
- 绞窄疝（急诊手术）

## 并发症
- **嵌顿**：疝内容物不能回纳
- **绞窄**：嵌顿后血供障碍，可导致肠坏死
- **肠梗阻**：肠管进入疝囊引起
- **复发**：术后可能复发

## 术后注意事项
- 避免重体力劳动3个月
- 预防便秘
- 控制慢性咳嗽
- 定期复查
