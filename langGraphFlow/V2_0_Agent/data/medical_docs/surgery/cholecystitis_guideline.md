# 急性胆囊炎诊断指南

## 概述
急性胆囊炎（Acute Cholecystitis）是胆囊的急性炎症，90%以上由胆囊结石引起。

## 常见症状
- **右上腹疼痛**：持续性疼痛，可向右肩背部放射
- **恶心、呕吐**：常见
- **发热**：体温38-39℃
- **黄疸**：部分患者出现（提示胆总管梗阻）

## 体征检查
- **右上腹压痛**：Murphy征阳性（深吸气时右上腹压痛加重）
- **右上腹肌紧张**
- **右上腹可触及肿大胆囊**（部分患者）
- **黄疸**：皮肤、巩膜黄染

## 辅助检查
1. **血常规**：白细胞计数升高，中性粒细胞比例升高
2. **肝功能**：转氨酶、胆红素可能升高
3. **腹部B超**：
   - 胆囊壁增厚（>3mm）
   - 胆囊增大
   - 胆囊内结石
   - 胆囊周围积液
4. **CT检查**：胆囊壁增厚、周围脂肪间隙模糊

## 诊断标准
满足以下条件：
1. 右上腹疼痛和压痛
2. 发热或白细胞升高
3. 影像学检查支持（B超或CT）

## 严重程度分级
- **轻度（1级）**：无器官功能障碍
- **中度（2级）**：合并其他疾病或器官功能障碍
- **重度（3级）**：出现器官功能衰竭

## 鉴别诊断
- 急性胰腺炎：血淀粉酶、脂肪酶升高，CT可见胰腺改变
- 消化性溃疡穿孔：X线可见膈下游离气体
- 右侧肺炎：胸片可见炎症
- 急性肝炎：肝功能明显异常

## 治疗原则
1. **保守治疗**（轻度）：
   - 禁食、补液
   - 抗生素治疗
   - 解痉止痛
2. **手术治疗**（中重度或保守治疗无效）：
   - 腹腔镜胆囊切除术（首选）
   - 开腹胆囊切除术
   - 胆囊造瘘术（高危患者）

## 手术时机
- **早期手术**：症状出现72小时内（推荐）
- **延期手术**：保守治疗6-12周后
- **急诊手术**：出现并发症（穿孔、坏疽）

## 并发症
- 胆囊穿孔
- 胆囊坏疽
- 胆汁性腹膜炎
- 胆总管结石
- 急性胰腺炎
