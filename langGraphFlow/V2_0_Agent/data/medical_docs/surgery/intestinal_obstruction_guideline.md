# 肠梗阻诊断指南

## 概述
肠梗阻（Intestinal Obstruction）是指肠内容物在肠道中运行受阻，是外科常见的急腹症。

## 分类
- **按病因**：机械性、动力性、血运性
- **按程度**：完全性、不完全性
- **按部位**：高位、低位
- **按血运**：单纯性、绞窄性

## 典型症状
- **腹痛**：阵发性绞痛，绞窄性为持续性剧痛
- **呕吐**：高位梗阻呕吐早、频繁，低位梗阻呕吐晚、量多
- **腹胀**：低位梗阻明显
- **停止排便排气**：完全性梗阻的特征

## 体征检查
- **腹部膨隆**：可见肠型和蠕动波
- **压痛**：单纯性轻度压痛，绞窄性明显压痛、反跳痛
- **肠鸣音**：机械性亢进，麻痹性减弱或消失
- **腹水征**：绞窄性梗阻可出现

## 辅助检查
1. **立位腹部X线**：
   - 可见多个气液平面
   - 阶梯状液平
   - 肠管扩张
2. **CT检查**：
   - 明确梗阻部位和原因
   - 判断是否绞窄
3. **血常规**：白细胞升高提示绞窄
4. **电解质**：低钾、低钠常见

## 诊断要点
1. 典型的"痛、吐、胀、闭"症状
2. 腹部X线可见气液平面
3. 肠鸣音改变

## 鉴别诊断
- 急性胃肠炎：有腹泻，无气液平面
- 急性胰腺炎：血淀粉酶升高
- 输尿管结石：绞痛，血尿
- 急性阑尾炎：转移性右下腹痛

## 治疗原则
1. **保守治疗**（单纯性、不完全性）：
   - 禁食、胃肠减压
   - 补液、纠正电解质紊乱
   - 抗生素预防感染
2. **手术治疗**（绞窄性、完全性、保守治疗无效）：
   - 解除梗阻
   - 切除坏死肠段
   - 肠吻合或造口

## 手术指征
- 绞窄性肠梗阻（急诊）
- 完全性肠梗阻
- 保守治疗24-48小时无效
- 出现腹膜炎征象

## 并发症
- 肠坏死
- 腹膜炎
- 感染性休克
- 水电解质紊乱
- 多器官功能衰竭

## 预后
- 单纯性肠梗阻：预后良好
- 绞窄性肠梗阻：及时手术预后较好，延误可危及生命
