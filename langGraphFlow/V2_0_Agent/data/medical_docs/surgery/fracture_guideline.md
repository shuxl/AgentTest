# 骨折诊断指南

## 概述
骨折是骨的完整性和连续性中断，是外科常见疾病。

## 分类
- **按原因**：外伤性、病理性
- **按程度**：完全性、不完全性
- **按形态**：横形、斜形、螺旋形、粉碎性

## 常见症状
- **疼痛**：局部剧痛，活动时加重
- **肿胀**：局部肿胀、瘀斑
- **畸形**：肢体变形、短缩
- **功能障碍**：不能活动
- **异常活动**：假关节活动
- **骨擦音**：可闻及骨摩擦音

## 体征检查
- **视诊**：畸形、肿胀、瘀斑
- **触诊**：压痛、骨擦感
- **动诊**：异常活动、功能障碍
- **量诊**：长度、角度测量

## 辅助检查
1. **X线检查**：明确骨折类型、移位情况
2. **CT检查**：复杂骨折、关节内骨折
3. **MRI**：软组织损伤、隐匿性骨折

## 诊断要点
1. 外伤史
2. 典型症状和体征
3. X线检查确诊

## 治疗原则
1. **复位**：手法复位或手术复位
2. **固定**：外固定或内固定
3. **功能锻炼**：早期、循序渐进

## 并发症
- 感染
- 骨不连
- 畸形愈合
- 关节僵硬
- 创伤性关节炎
