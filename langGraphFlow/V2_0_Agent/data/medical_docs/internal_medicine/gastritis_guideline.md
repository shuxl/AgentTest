# 胃炎诊断与治疗

## 概述
胃炎（Gastritis）是指胃黏膜的炎症性疾病，分为急性和慢性胃炎。

## 常见症状
- 上腹痛、不适
- 腹胀、嗳气
- 恶心、呕吐
- 食欲不振
- 反酸、烧心
- 消化不良

## 急性胃炎
- 起病急，症状明显
- 常见原因：药物、酒精、应激、感染
- 治疗：去除病因，对症治疗

## 慢性胃炎
- 起病缓慢，症状反复
- 常见原因：幽门螺杆菌感染、自身免疫、胆汁反流
- 分类：浅表性胃炎、萎缩性胃炎

## 诊断方法
1. **胃镜检查**：观察胃黏膜病变
2. **幽门螺杆菌检测**：C13/C14呼气试验、快速尿素酶试验
3. **病理活检**：明确病变性质

## 幽门螺杆菌感染
- 是慢性胃炎的主要病因
- 与消化性溃疡、胃癌相关
- 根除方案：PPI + 两种抗生素 + 铋剂

## 治疗原则
1. **去除病因**：根除Hp、停用损伤药物
2. **抑酸治疗**：PPI、H2受体拮抗剂
3. **保护胃黏膜**：铋剂、铝制剂
4. **对症治疗**：促动力药、消化酶

## 饮食建议
- 规律饮食，少食多餐
- 避免辛辣、刺激性食物
- 戒烟限酒
- 避免过冷过热食物
