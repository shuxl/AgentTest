# 冠心病诊断要点

## 概述
冠心病（Coronary Heart Disease）是指冠状动脉粥样硬化导致的心肌缺血性疾病。

## 典型症状
- 胸痛：胸骨后压榨性疼痛
- 心绞痛：活动后出现，休息后缓解
- 胸闷、气短
- 心悸、心律失常
- 乏力、出汗

## 危险因素
- 年龄：男性>45岁，女性>55岁
- 高血压
- 糖尿病
- 高脂血症
- 吸烟
- 家族史
- 肥胖

## 诊断方法
1. **心电图**：ST段改变、T波倒置、病理性Q波
2. **运动负荷试验**：评估运动耐量
3. **冠状动脉CTA**：无创检查
4. **冠状动脉造影**：金标准

## 分型
1. **稳定型心绞痛**：症状稳定，可预测
2. **不稳定型心绞痛**：症状加重或新发
3. **急性心肌梗死**：持续胸痛，心肌酶升高

## 治疗原则
1. **药物治疗**：
   - 抗血小板：阿司匹林、氯吡格雷
   - 他汀类：降脂、稳定斑块
   - β受体阻滞剂：降低心率、心肌耗氧
   - ACEI/ARB：改善预后

2. **介入治疗**：PCI（经皮冠状动脉介入）
3. **外科治疗**：CABG（冠状动脉旁路移植术）

## 预防措施
- 控制危险因素
- 规律服药
- 定期复查
- 健康生活方式
