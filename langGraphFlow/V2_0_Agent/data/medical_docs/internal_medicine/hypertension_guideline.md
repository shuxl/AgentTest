# 高血压诊断指南

## 概述
高血压（Hypertension）是指血压持续升高的慢性疾病，是心血管疾病的重要危险因素。

## 诊断标准
1. **正常血压**：收缩压 < 120 mmHg 且 舒张压 < 80 mmHg
2. **高血压前期**：收缩压 120-139 mmHg 或 舒张压 80-89 mmHg
3. **1级高血压**：收缩压 140-159 mmHg 或 舒张压 90-99 mmHg
4. **2级高血压**：收缩压 ≥ 160 mmHg 或 舒张压 ≥ 100 mmHg

## 常见症状
- 头痛，特别是早晨起床时
- 头晕、眩晕
- 心悸、胸闷
- 视力模糊
- 耳鸣
- 疲劳、乏力

## 诊断要点
1. 至少测量3次不同时间的血压
2. 排除白大衣高血压
3. 进行24小时动态血压监测（必要时）
4. 评估靶器官损害情况

## 鉴别诊断
需要排除继发性高血压：
- 肾性高血压
- 内分泌性高血压
- 药物性高血压

## 治疗原则
1. 生活方式干预：低盐饮食、适量运动、控制体重
2. 药物治疗：ACEI、ARB、CCB、利尿剂等
3. 定期监测血压，调整治疗方案
