# 肺炎诊断要点

## 概述
肺炎（Pneumonia）是指终末气道、肺泡和肺间质的炎症，可由细菌、病毒、真菌等引起。

## 常见症状
- 发热、寒战
- 咳嗽、咳痰（脓性痰或铁锈色痰）
- 胸痛、呼吸困难
- 乏力、食欲不振
- 头痛、肌肉酸痛

## 细菌性肺炎
- 起病急，高热
- 咳嗽、咳脓痰
- 常见病原体：肺炎链球菌、金黄色葡萄球菌
- 血白细胞升高

## 病毒性肺炎
- 起病较缓
- 干咳为主
- 常见病原体：流感病毒、腺病毒
- 血白细胞正常或降低

## 诊断方法
1. **胸部X线**：肺部浸润影
2. **血常规**：白细胞、中性粒细胞比例
3. **痰培养**：明确病原体
4. **血培养**：重症患者
5. **CT检查**：复杂病例

## 严重程度评估
- **轻症**：门诊治疗
- **重症**：需要住院，满足以下任一条件：
  - 意识障碍
  - 呼吸频率 ≥ 30次/分
  - 血压 < 90/60 mmHg
  - 氧合指数 < 250

## 治疗原则
1. **抗感染治疗**：
   - 社区获得性肺炎：β-内酰胺类 + 大环内酯类
   - 医院获得性肺炎：根据药敏选择
2. **对症支持**：退热、止咳、化痰、吸氧
3. **并发症处理**：胸腔积液、呼吸衰竭

## 预防措施
- 接种肺炎疫苗
- 增强免疫力
- 避免接触感染源
- 戒烟
