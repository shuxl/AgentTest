# 糖尿病诊断与治疗

## 概述
糖尿病（Diabetes Mellitus）是一组以高血糖为特征的代谢性疾病，主要分为1型和2型糖尿病。

## 诊断标准
满足以下任一标准即可诊断：
1. 空腹血糖 ≥ 7.0 mmol/L
2. 随机血糖 ≥ 11.1 mmol/L
3. OGTT 2小时血糖 ≥ 11.1 mmol/L
4. HbA1c ≥ 6.5%

## 常见症状
- 多饮、多尿、多食
- 体重下降
- 视力模糊
- 疲劳、乏力
- 伤口愈合缓慢
- 反复感染

## 1型糖尿病特点
- 多见于青少年
- 起病急，症状明显
- 胰岛素绝对缺乏
- 需要胰岛素治疗

## 2型糖尿病特点
- 多见于中老年
- 起病缓慢，症状不明显
- 胰岛素抵抗为主
- 可通过口服药物控制

## 并发症
- 糖尿病肾病
- 糖尿病视网膜病变
- 糖尿病神经病变
- 糖尿病足
- 心血管疾病

## 治疗目标
- 空腹血糖：4.4-7.0 mmol/L
- 餐后2小时血糖：< 10.0 mmol/L
- HbA1c：< 7.0%

## 治疗方案
1. 生活方式干预：饮食控制、运动、体重管理
2. 口服降糖药：二甲双胍、磺脲类、DPP-4抑制剂等
3. 胰岛素治疗：适用于1型糖尿病和部分2型糖尿病
