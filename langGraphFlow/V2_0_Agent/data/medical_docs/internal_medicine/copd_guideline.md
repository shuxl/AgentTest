# 慢性阻塞性肺疾病（COPD）诊断

## 概述
COPD是一种以持续气流受限为特征的疾病，与有害气体或颗粒的异常炎症反应有关。

## 主要症状
- 慢性咳嗽：晨起明显
- 咳痰：白色黏液痰或脓性痰
- 气短、呼吸困难：活动后加重
- 喘息、胸闷

## 危险因素
- 吸烟：最重要的危险因素
- 职业粉尘和化学物质
- 空气污染
- 感染：反复呼吸道感染
- 遗传因素：α1-抗胰蛋白酶缺乏

## 诊断标准
1. **症状**：慢性咳嗽、咳痰、呼吸困难
2. **危险因素暴露史**：吸烟、职业暴露
3. **肺功能检查**：FEV1/FVC < 70%（吸入支气管扩张剂后）

## 严重程度分级
- **GOLD 1级（轻度）**：FEV1 ≥ 80%预计值
- **GOLD 2级（中度）**：FEV1 50-79%预计值
- **GOLD 3级（重度）**：FEV1 30-49%预计值
- **GOLD 4级（极重度）**：FEV1 < 30%预计值

## 鉴别诊断
- 支气管哮喘：可逆性气流受限
- 支气管扩张：反复感染、大量脓痰
- 充血性心力衰竭：心源性呼吸困难

## 治疗原则
1. **戒烟**：最重要的干预措施
2. **支气管扩张剂**：
   - 短效：沙丁胺醇、异丙托溴铵
   - 长效：噻托溴铵、沙美特罗
3. **糖皮质激素**：中重度患者
4. **氧疗**：长期家庭氧疗（LTOT）
5. **肺康复**：运动训练、呼吸训练

## 急性加重期处理
- 增加支气管扩张剂使用
- 全身糖皮质激素
- 抗生素（有感染征象时）
- 无创通气（必要时）
